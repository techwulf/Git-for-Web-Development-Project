1. What is Semantic HTML? 
  Descriptive tags within HTML that help communicate the meaning of it's
  elements.

2. What is HTML used for? 
  The framework of a website. It describes everything that will be on a
  page.

3. What is an attribute and where do we put it? 
  Attributes are items placed in an element that contain useful
  information for and/or about the element.

4. What is the h1 tag used for? How many times should I use it on a page?
  h1 tags are typically used for main headers on a page. Typically only
  used once.

5. Name two tags that have required attributes
  <img> <a>

6. What do we put in the head of our HTML document? 
  links to external stylesheets, the title, anything that applies to the
  page, but isn't explicitly in it except scripts.

7. What is an id?

8. What elements can I add an id to? 
  Any elements.

9. How many times can I use the same id on a page?

10. What is a class? 
  Another special marker for a element.

11. What elements can I add a class to?

12. How many times can I use the same class on a page? 
  Infinite

13. How do I get my link to open in a new tab?

14. What is the alt attribute in the image tag used for?

15. How do I reference an id?

17. What is CSS used for?

  want to apply a specific style to that specific child of the parent.

24. What is a pseudo selector?

  is applied.

Stretch Questions

1. What is the difference between an inline element and a block element?
  A block element takes up the whole width of it's container, and can be 
  resized vertically. An inline element only takes up the space it needs
  to exist.

2. What happens when an element is positioned absolutely? 
  It will exist specifically where you tell it to within it's container
  regardless of what order the other elements are in the same container.

3. How do I make an element take up only the amount of space it needs but also have the ability to give it a width? 
  Change it to display as an inline-block element.

4. Name 3 elements that are diplay block by default, 2 elements that are display inline by default and 1 element that is display inline-block by default
  h1 p div are block level. a span strong are inline elements. img is an inline
  block element.

5. In your own words, explain the box model. What is the "fix" for the box model, in other words, how do we make all elements respect the width we've given them? 
  The box model is how all the elements are contained within eachother. Flexbox
  can be used to overcome allot of the issues when you want elements to appear
  where and how you want them to.
